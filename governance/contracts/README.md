# Contracts

This sector stores explicit interaction and governance contracts that shape how human-model relation is represented inside the repository.

## Scope

Files here should describe:
- interaction protocols,
- relational-formal constraints,
- truth/clarity/distortion rules,
- operator-level behavioral invariants for Sapiens-facing layers.

## Distinction from orbital architecture

These contracts are not the same thing as the orbital source architecture.

- orbital architecture describes repository-scale coherence geometry and diagnostics,
- governance contracts describe how human-model interaction should be constrained,
- Sapiens-facing code may depend on both.
