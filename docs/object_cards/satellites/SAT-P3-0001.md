## Semantic Mass — M_sem

`M_sem(f) = α·M_EC + β·M_ZS + χ·C_dep + δ·C_prov + ε·C_exec`

| Komponent | Znaczenie | Proxy |
|-----------|-----------|-------|
| M_EC | Euler-Collatz affinity | info_mass × coupling_sum |
| M_ZS | Zeta-Schrödinger resonance | coherence_weight × tau_norm |
| C_dep | dependency centrality | Σ coupling weights do węzła |
| C_prov | provenance depth | amplitude / horizon_class |
| C_exec | execution activity | amplitude / coupling_ciel |

**Kepler-like orbit period:** `T² = a³ / M_sem` gdzie `a = poincare_radius(θ)`

## Subjective Time — Δτ

`Δτ_i = Δt · g(r, C, Δ, m, A) = (1/τ_ref) · (1−r²) · C/(1+Δ) · √(m·A)`

- Encje bliżej centrum dysku (r→0): szybszy czas subiektywny
- Encje przy granicy (r→1): wolniejszy czas (dylatacja hiperboliczna)
- `τ_ref = 0.353` (środkowa wartość własna z relational_contract.yaml)

Winding rate: `w_rate = 1 / (T_orbit · τ_scale)`

Najszybszy czas subiektywny (2026-04-30): `constraints` (τ=2.19, r=0.27)
