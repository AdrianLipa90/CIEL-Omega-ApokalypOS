# Integration docs sector

This directory is the target home for human-readable integration notes and addenda.

## Scope

Files here should describe:
- upstream bindings,
- shell-versus-engine distinctions,
- orbital integration notes,
- bridge-layer integration rules,
- import interpretation.

## Phase-1 migration rule

Legacy integration documents may still remain in flat `docs/` paths during early migration.
This sector establishes the final documentation geometry first.
