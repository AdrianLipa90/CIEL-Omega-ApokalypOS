# Archive sector

This directory is the target home for transitional, superseded, or explicitly historical documentation artifacts.

## Scope

Files that should eventually live here include:
- transition plans,
- temporary addenda later merged elsewhere,
- micro-audits tied to a specific migration phase,
- preserved historical notes that are no longer canonical navigation entrypoints.

## Migration rule

Do not archive files merely because they are old.
Archive them when they are no longer primary navigation documents but still useful as historical context.
