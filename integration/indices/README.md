# Indices sector

This directory is the target home for machine-readable index and cross-reference layers.

## Scope

Files here should describe navigation and reference structures, for example:
- hyperspace indices,
- cross-reference maps,
- machine-readable path registries,
- lookup-oriented integration indices.

## Migration note

During refactor, legacy index files may still exist directly under `integration/`.
This sector establishes the target geometry first.

## Excluded

This sector is not for:
- reports,
- live snapshots,
- protected orbital runtime internals,
- runtime executables.
