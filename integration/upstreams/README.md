# Upstreams sector

This directory is the target home for machine-readable upstream binding state.

## Scope

Files here should describe external or upstream repository relations, for example:
- upstream inventories,
- shell maps,
- live upstream binding state,
- upstream synchronization descriptors.

## Migration note

During refactor, legacy upstream files may still exist directly under `integration/`.
This sector establishes the target geometry first.

## Excluded

This sector is not for:
- local reports,
- native executable logic,
- protected orbital runtime packages,
- broad archival notes.
