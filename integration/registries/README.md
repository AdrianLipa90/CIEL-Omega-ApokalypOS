# Registries sector

This directory is the target home for stable machine-readable registry files.

## Scope

Files here should describe named repository objects and persistent identity-like structures, for example:
- repository registries,
- object registries,
- index registry mappings,
- stable machine-readable inventories.

## Migration note

During refactor, legacy registry files may still exist directly under `integration/`.
This sector establishes the target geometry first.

## Excluded

This sector is not for:
- live runtime reports,
- temporary snapshots,
- protected orbital runtime files,
- executable code.
